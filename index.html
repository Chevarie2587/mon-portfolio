Agent vocal IA – Prise de rendez-vous

AI Voice Agent – Appointment Booking

Dans le cadre d’un projet personnel, j’ai développé un assistant vocal basé sur Python capable d’enregistrer une demande orale, de la transcrire automatiquement avec Whisper, puis d’en extraire l’intention (jour, heure, type de rendez-vous) à l’aide d’un modèle GPT (via l’API OpenRouter) tout en se structurant selon des contraintes pré-déterminées. L’événement est ensuite ajouté dynamiquement à Google Calendar à l’aide de l’API officielle. L’agent vocal confirme le rendez-vous par synthèse vocale avec pyttsx3.

As a personal project, I developed a voice assistant based on Python capable of recording a spoken request, automatically transcribing it with Whisper, then extracting the intent (day, time, type of appointment) using a GPT model (via the OpenRouter API), while applying pre-defined constraints. The event is then dynamically added to Google Calendar using the official API. The voice agent confirms the appointment through speech synthesis using pyttsx3.

Ce projet m’a permis d’explorer l’intégration de plusieurs technologies : reconnaissance vocale (SpeechRecognition), IA générative, traitement de texte en langage naturel, OAuth2 Google et gestion d’un flux conversationnel voix-à-voix. Il est conçu pour être exécuté localement de façon sécurisée, sans dépendance à un service payant comme l’API OpenAI.

This project allowed me to explore the integration of several technologies: voice recognition (SpeechRecognition), generative AI, natural language processing, Google OAuth2, and real-time voice-to-voice conversation flow management. It is designed to run securely on a local machine, without relying on paid services like the OpenAI API.

Mon intention est de réutiliser cette logique et l’adapter à des réalités concrètes d’entreprise, par exemple pour automatiser la prise de rendez-vous clients, les réponses à des appels entrants ou la création d'événements internes.

My intention is to reuse this logic and adapt it to real business use cases, such as automating client appointment scheduling, answering incoming calls, or creating internal events.
